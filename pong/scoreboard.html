Scoreboard

Alright, the scoreboard is probably the simplest component of this particular pong game. The main thing you'll need is a pair of variables that keep track of each player's scores. These scores can be updated by the ball when it leaves the screen, since that's the scoring condition. Of course, this can get more complicated as the development of your game goes on, depending on how many players you want to add, how many balls you add to your game, and so on. But just keeping score in mind, let's go ahead and set up a class for your scoreboard.

Setting up the Class

(If you need a refresher on the syntax of making classes in p5.js, there's an example in the documentation)

Lets first name the class using function Scoreboard(){}, and then fill in its scope with descriptive functions and properties. Since there's only two variables to keep track of you can make two numbers and set their initial value to zero. You'll then need a way to show the score, which is where the draw function comes in.

 function ScoreBoard()
{
  this.p1Score = 0;
  this.p2Score = 0;
   this.draw = function(){};
}

Since the manipulation of the score will be handled by the ball, there isn't a lot you need to do with p1Score and p2Score just yet. For now, you can fill in the draw function. The value of each score variable can be displayed on the screen with p5's text() function.

 this.draw = function()
 {
   textSize(24);
   fill(255);
   text(this.p1Score, width-75, 25);
   text(this.p1Score, width+75, 25);
 };
}

In this example, the position of each score is set to 75 pixels on either side of the center of the x-axis, and 25 pixels from the top. Maybe you want them to be on the left and right edges of the screen?
